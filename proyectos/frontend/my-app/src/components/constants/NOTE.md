# Sobre las constantes utilizadas en la aplicación
Este archivo contiene una lista de constantes que se utilizan en varias partes de la aplicación para mantener la coherencia y facilitar el mantenimiento del código.
Estas son temporales, y van a cambiar por el uso de una base de datos real en el futuro.
Estos datos reales eran obtenidos por medio de middleware entre la BD y el frontend, pero por limitaciones de tiempo y complejidad, se decidió usar datos estáticos por ahora.
Se ampliaria en una proxima iteración del proyecto.
